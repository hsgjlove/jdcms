简单CMS - 专业影视管理系统
简单CMS是一款功能强大的影视管理系统，具有以下核心功能：

强大的影视获取功能，无需采集只需要一个采集接口即可获取资源站全部影视
全自动分类管理，自动获取源站所有影视分类
高性能缓存系统，优化分类缓存，提升访问速度
智能过滤系统自动过滤空分类，通过后台设定过滤规则过滤对应分类，自动检测敏感分类并屏蔽分类和列表，自动识别设定的敏感分类超过三个不同的敏感分类接口会被屏蔽，保护网站安全
多格式接口支持，包括XML、xmlsea、JSON、VOD格式
强大的后台管理，自定义无限增加删除接口以及资源倒序，资源名称，按钮颜色，过滤规则、自动获取用户网站网址，支持自定义Logo、外链，网站名，解析，统计，音乐播放器和版权信息，密码修改等
播放页支持上下集播放，支持评分评论，播放列表支持直连播放和m3u8解析播放
无需数据库，精美后台设计，数据保存在后台
前端添加清理缓存功能，自动识别用户设定的资源站名称可清理对应接口数据，分类，以及删除全部接口数据
前端强大的历史记录功能自动记录用户播放的影视剧集，点击历史记录会自动跳转到视频详情页自动播放精准到那一集
简单CMS让您轻松搭建专业的影视网站，享受高效的管理体验。
